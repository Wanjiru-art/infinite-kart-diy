WEBSITE DESCRIPTION
================================================================
Infinite Kart DIY is a fully functional multi-page e-commerce 
website for a paint by numbers business based in Nairobi, Kenya.
The website allows customers to browse and purchase paint by 
numbers kits online.

================================================================
PAGES INCLUDED
================================================================
1. index.html    - Home Page
2. about.html    - About Us Page
3. products.html - Products Page
4. services.html - Services Page
5. contact.html  - Contact Us Page

================================================================
TECHNOLOGIES USED
================================================================
- HTML5
- CSS3 (Flexbox, Grid, Animations, Media Queries)
- JavaScript (Form Validation, Add to Cart, Back to Top)
- Font Awesome Icons
- Google Fonts
- GitHub Pages (Hosting)

================================================================
JAVASCRIPT FEATURES
================================================================
1. Contact Form Validation
2. Add to Cart Notification
3. Back to Top Button

================================================================
HOW TO RUN THE WEBSITE
================================================================
Option 1 - Live Website:
Visit: https://infinite-kart-diy.netlify.app/

Option 2 - Local:
1. Download or clone the repository
2. Open index.html in any modern browser
3. Or use VS Code with Live Server extension
